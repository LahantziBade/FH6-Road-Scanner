# 地平线道路扫描器 / FH6 Road Scanner

**语言 / Language / 言語：**
简体中文 | [English](./README.en.md) | [日本語](./README.jp.md)

一个用于《极限竞速：地平线6》地图道路探索的辅助工具。
它通过模拟鼠标逐行移动，并截图检测左下角“快速移动”按钮是否消失，帮助玩家定位可能遗漏的未探索道路。

## 快速开始

前往[Releases](https://github.com/LahantziBade/FH6-Road-Scanner/releases/latest)下载最新版本压缩包，解压即可使用。

## 功能

- 自动逐行扫描地图
- 检测疑似未探索道路
- 命中后自动截图、蜂鸣并停止
- 支持 F8 快捷键停止扫描
- 支持自定义扫描区域、检测区域、扫描步长和阈值

## 使用说明

1. 打开游戏地图。
2. 将鼠标放在已探索道路上，确认左下角出现“快速移动”按钮。
3. 打开程序，点击“截取/更新模板”。
4. 点击“测试当前差异分数”，确认分数正常。
5. 点击“开始扫描”，切回游戏等待自动扫描。
6. 扫描中可按 F8 停止。

## 原理说明

本工具不会修改游戏文件，不读取游戏内存，不注入进程，也不会修改存档或联网数据。

它只是模拟鼠标移动，并通过截图识别界面变化，相当于把人工“晃鼠标找路”的过程自动化。

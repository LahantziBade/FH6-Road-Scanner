# 地平線道路スキャナー / FH6 Road Scanner

**言語 / Language / 语言:**
[简体中文](./README.md) | [English](./README.en.md) | 日本語

*Forza Horizon 6* のマップ道路探索を補助するツールです。

マップ上でマウスを行ごとに自動移動し、画面左下の「ファストトラベル」ボタンが消えるかどうかをスクリーンショットで検出することで、見落としている可能性のある未探索道路を探しやすくします。

## クイックスタート

[Releases](https://github.com/LahantziBade/FH6-Road-Scanner/releases/latest) ページから最新版の圧縮ファイルをダウンロードし、解凍してそのまま使用できます。

## 機能

* マップを行ごとに自動スキャン
* 未探索の可能性がある道路を検出
* 検出時に自動でスクリーンショットを保存し、通知音を鳴らして停止
* F8 キーによるスキャン停止に対応
* スキャン範囲、検出範囲、スキャン間隔、しきい値をカスタマイズ可能

## 使い方

1. ゲーム内マップを開きます。
2. すでに探索済みの道路にマウスを置き、左下に「ファストトラベル」ボタンが表示されることを確認します。
3. プログラムを開き、「テンプレートを取得 / 更新」をクリックします。
4. 「現在の差分スコアをテスト」をクリックし、スコアが正常な範囲であることを確認します。
5. 「スキャン開始」をクリックし、ゲーム画面に戻って自動スキャンの開始を待ちます。
6. スキャン中は F8 キーで停止できます。

## 仕組み

このツールは、ゲームファイルの変更、ゲームメモリの読み取り、プロセスへの注入、セーブデータやオンラインデータの変更を一切行いません。

マウス移動をシミュレートし、スクリーンショットによって画面表示の変化を検出するだけです。つまり、手動で行う「マウスを動かして道路を探す方法」を自動化したものです。
